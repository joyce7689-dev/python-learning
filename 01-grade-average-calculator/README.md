# Grade Average Calculator

## Description

This project is a Python command-line application that allows users to enter school subjects and grades.

The program calculates the average grade, identifies passed and failed subjects, and displays the highest and lowest grades.

## Features

- Add multiple subjects
- Validate user input
- Calculate the average grade
- Identify passed and failed subjects
- Display highest and lowest grades

## Concepts Practiced

- Variables
- Lists
- Functions
- Loops
- Conditional Statements
- Exception Handling
- Input Validation

## Technologies

- Python 3
